About This Platform

For Everyone

Whether you're a patient, student, doctor, or just curious about health, this platform is built to help anyone understand risk factors and make smarter choices.

Simple & Clear

You don't need any medical or tech background. Just enter your info, and get instant risk scores, symptom checks, and clear explanations - no jargon.

Transparent AI

We use trained CatBoost, XGBoost, and EfficientNet models and return clear prediction labels, probabilities, and confidence so results are easier to interpret.

Tech Stack

Frontend: HTML + Tailwind
Backend: FastAPI + Uvicorn
ML: CatBoost, XGBoost, EfficientNet-B0
Runtime: Joblib + PyTorch (.pth)
Production-style API + model metadata
